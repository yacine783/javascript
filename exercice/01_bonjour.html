bonjour

  • creez un fichier bonjour.html et faites-le pointer vers un fochier bonjour.js
  • ecrire le code invitant l'uutilisateur a saisir d'abord son nom et ensuite son prenom dans 2 boites de dialogue successives
  • afficher ensuites dans un autre boite de dialogue le nom et le prenom de l'utilisateur
  • attention a bien respecter les espaces entre les mots dans le meme message finale.
